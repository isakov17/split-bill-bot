# 📊 SplitBillBot — Бот для совместного расчета трат

Telegram-бот на Python (aiogram 3), который помогает компаниям друзей честно делить расходы в поездках, кафе или на мероприятиях. 

## ✨ Основные функции
- **🤖 Умное добавление:** Фиксация трат через текст (напр., `+500 такси`) или через принудительное добавление друзей (свайп + `/add`).
- **🧾 Сканирование чеков (OCR):** Распознавание QR-кодов чеков через API. Бот парсит товары и запускает интерактивную карусель для распределения позиций между участниками.
- **📈 Минимизация транзакций:** Алгоритм автоматически "схлопывает" долги, чтобы вместо множества мелких переводов каждый совершил 1-2 крупных.
- **🛡 Двухфакторное подтверждение:** После оплаты должник нажимает «Оплатил» в личке бота, а кредитор должен подтвердить получение средств в своей личке.
- **📱 Личный кабинет:** Индивидуальный список долгов с реквизитами (СБП) в приватном чате с ботом.
- **📊 Экспорт:** Выгрузка всей истории трат в формате CSV для детального анализа.

## 🛠 Технологии
- **Язык:** Python 3.12+
- **Фреймворк:** Aiogram 3.x
- **База данных:** SQLite (aiosqlite)
- **OCR API:** proverkacheka.com
