# 📊 SplitBillBot — Умный расчет совместных трат

Telegram-бот на **aiogram 3**, разработанный для автоматизации разделения расходов в путешествиях и компаниях друзей.

## 🚀 Основные возможности
* **🧾 OCR-сканирование чеков:** Интеграция с API для распознавания QR-кодов. Карусель товаров позволяет за секунды распределить позиции чека между участниками.
* **⚙️ Алгоритм оптимизации долгов:** Бот сводит количество транзакций к минимуму (минимизация графа платежей).
* **🔐 Двухфакторная оплата:** Система подтверждения переводов обеими сторонами через личные сообщения.
* **📂 Экспорт данных:** Возможность выгрузки истории транзакций в формат CSV.
* **🛠 Гибкое управление:** Принудительное добавление участников через Reply-команды и поддержка пресетов с реквизитами.

## 🛠 Установка и запуск (на VPS)

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/isakov17/split-bill-bot.git](https://github.com/isakov17/split-bill-bot.git)
   cd split-bill-bot
   ```
2. **Настройте окружение:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
3. **Настройте переменные окружения:**
   Создайте файл `.env` и добавьте туда:
   - `BOT_TOKEN`
   - `PROVERKA_TOKEN` (API от proverkacheka.com)

4. **Запуск через systemd:**
   Инструкция по настройке сервиса находится в файле `splitbot.service`.

## 📋 Команды
- `/join` — стать участником расчетов.
- `/add` — добавить трату (напр. `+500 еда`) или участника (через reply).
- `/calc` — финальный расчет и схема переводов.
- `/help` — подробная инструкция для пользователей.
```

### 3. Обновление и синхронизация проекта

Поскольку твой `splitbot.service` лежит в системной папке `/etc/systemd/system/`, а проект — в `/root/project/split-bill-bot/`, давай сделаем так, чтобы актуальная версия сервиса тоже была в гите.

**Выполни эти команды на сервере:**

```bash
# 1. Копируем актуальный сервис из системы в папку проекта для Гита
cp /etc/systemd/system/splitbot.service /root/project/split-bill-bot/

# 2. Создаем файл README.md (если еще не создал)
nano /root/project/split-bill-bot/README.md
# (Вставляешь текст из пункта 2 выше, нажимаешь Ctrl+O, Enter, Ctrl+X)

# 3. Добавляем изменения в Гит
git add bot.py README.md splitbot.service
git commit -m "Add help command and documentation"
git push origin main
```
