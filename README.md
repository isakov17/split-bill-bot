# 📊 SplitBillBot — Умный расчет совместных трат в Telegram

**SplitBillBot** — это продвинутый бот на базе `aiogram 3` и `aiosqlite`, разработанный для автоматизации разделения расходов в путешествиях, ресторанах и компаниях друзей. Больше никаких долгих подсчетов в заметках — бот сам сканирует чеки и оптимизирует долги.

## ✨ Основные возможности

* **🔍 OCR (Распознавание чеков):** Отправьте фото чека с QR-кодом, и бот автоматически распарсит его на отдельные позиции.
* **🎠 Интерактивная карусель:** Удобный интерфейс для распределения товаров из чека между участниками «в один клик».
* **⚖️ Оптимизация транзакций:** Алгоритм автоматически «схлопывает» цепочки долгов, сводя количество переводов к минимуму.
* **🛡 Двухфакторное подтверждение (2FA):** Должник отмечает оплату, кредитор подтверждает приход денег. Только после этого баланс официально обновляется.
* **📱 Личный кабинет:** Прямо в Telegram бот присылает список ваших долгов с именами и реквизитами (СБП).
* **📂 Экспорт данных:** Возможность выгрузить всю историю поездки в CSV-файл (Excel).

## 🚀 Команды бота

| Команда | Описание |
| :--- | :--- |
| `/join` | Присоединиться к текущему чату (группе) |
| `/add` | Добавить трату (напр. `+500 такси`) или друга (через ответ на сообщение) |
| `/calc` | Вывести итоги: кто, кому и сколько должен перевести |
| `/help` | Открыть подробную инструкцию пользователя |
| `/me` | Посмотреть свой личный баланс в этом чате |
| `/history` | Посмотреть 10 последних трат с возможностью удаления |
| `/export` | Выгрузить детальный отчет в формате Excel (CSV) |
| `/reset` | Полная очистка данных и завершение текущей поездки |

## 🛠 Техническая настройка (для VPS)

### 1. Требования
* Python 3.12+
* Библиотеки: `aiogram`, `aiosqlite`, `python-dotenv`, `aiohttp`

### 2. Установка
```bash
git clone [https://github.com/isakov17/split-bill-bot.git](https://github.com/isakov17/split-bill-bot.git)
cd split-bill-bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Конфигурация (.env)
Создайте файл `.env` в корне проекта:
```env
BOT_TOKEN=ваш_токен_от_BotFather
PROVERKA_TOKEN=ваш_токен_от_proverkacheka.com
```

### 4. Автозапуск (Systemd)
Файл сервиса должен быть создан по пути /etc/systemd/system/splitbot.service.

Содержимое splitbot.service:

```splitbot.service
Description=Split Bill Telegram Bot
After=network.target

[Service]
User=root
Group=root
WorkingDirectory=/root/project/split-bill-bot
# Убедись, что скрытый файл .env с токеном лежит в этой же папке!
EnvironmentFile=/root/project/split-bill-bot/.env
ExecStart=/root/project/split-bill-bot/venv/bin/python /root/project/split-bill-bot/bot.py
Restart=always
RestartSec=5
Environment=PYTHONUNBUFFERED=1

[Install]
WantedBy=multi-user.target
```

### Команды для активации:

```bash
sudo systemctl daemon-reload
sudo systemctl enable splitbot.service
sudo systemctl start splitbot.service
```
## 📖 Инструкция для пользователей

* **Начало:** Все участники пишут `/join` в группе.
* **Траты:**
    * Пишите текстом: `+1200 Вино` или `/add 800 Пицца`.
    * Или скиньте фото чека и подпишите его словом **«чек»**.
* **Расчет:** В конце вечера напишите `/calc`.
* **Оплата:**
    1. Нажмите кнопку **«💳 Мои долги»**.
    2. Перейдите в личку с ботом, посмотрите реквизиты и переведите деньги.
    3. Нажмите **«✅ Оплатил»**. После этого получатель должен будет подтвердить приход денег в своей личке.
